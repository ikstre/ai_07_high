  - `fcntl`이 없는 환경에서는 조건부 import로 처리하고, 기본 `GPU_WORKER_MODE=always_on` 테스트 경로가 정상 동작하도록 했다.

## 상품 입력 상태 동기화 수정

- 광고 카드 미리보기가 기본값처럼 보이는 원인을 확인했다.
- 기존 상품 정보 입력 위젯은 `st.session_state.product_name = st.text_input(...)`처럼 반환값을 다시 대입하는 방식이었다.
- 일부 위젯, 특히 `판매 채널` selectbox는 명시적인 `key`와 현재값 기반 `index`가 없어 rerun 시 기본 옵션으로 돌아갈 수 있었다.
- `product_type`, `product_name`, `price`, `target_channel`, `target_customer`, `selling_point` 위젯을 명시적인 `key` 기반으로 변경했다.
- 상품 입력값이 바뀌면 기존 광고 문구, 포스터, 실사 이미지 작업 결과를 무효화하도록 `invalidate_generated_ad_outputs()`를 추가했다.
