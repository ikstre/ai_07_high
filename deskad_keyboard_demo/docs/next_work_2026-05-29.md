## 2. P2 — 기능 확장 (회귀 픽스 후)

### 2-1. M2 — image job batch 생성

- POST `/ai/image/jobs` 에 `batch_size` 파라미터 추가 (1-4)
- ComfyUI workflow 를 batch_size 만큼 큐잉 (seed 만 다르게)
- job 상태에 `children: [child_job_id, ...]` 추가
- UI (U4 갤러리) 와 자연스럽게 합쳐짐

### 2-2. M3 — `/ai/image/jobs/{id}/regenerate` 엔드포인트

- 같은 payload + 새 seed 로 단발 변주
- M2 의 batch 와 의도 분리: regenerate 는 "한 번 더 (다른 결과)", batch 는 "한 번에 N장"

### 2-3. U2 — 한국 모델 비교 4-card grid

- `/ai/copy/experiment` 결과를 expander 1개 안에 세로로 쌓는 대신 4-card grid (kanana / midm / local / fallback)
- 각 카드: provider 라벨, headline, subcopy, hashtags, error (있으면)

### 2-4. M4 — 모델 hot-swap

- `OLLAMA_MODEL` 을 .env 가 아닌 런타임에 변경 가능
- `/ai/providers` 에 사용 가능한 모델 목록 + 현재 선택 표시
- Streamlit 사이드바에서 변경
